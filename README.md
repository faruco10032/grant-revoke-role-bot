# Grant Revoke Role Bot

一時的にDiscordロールを付与するBotです。指定した時間（分・日）が経過すると、ロールは自動で剥奪されます。

## 主な機能

- `/setup_button` コマンドでロール付与ボタンを生成（管理者のみ）
- ユーザーがボタンを押すとロールが付与され、指定時間後に自動で剥奪
- 分単位・日単位での期間指定に対応（併用可）
- 通知先チャンネルを指定して、ロール剥奪時にメンション通知（オプション）
- Bot再起動後もタイマー・ボタンが継続動作（JSON永続化）
- `/help` コマンドで使い方を表示

## セットアップ

1. `.env` ファイルを用意し、以下のように記述：

   ```env
   DISCORD_TOKEN=あなたのBotトークン
   ```

2. 依存パッケージをインストール：

   ```bash
   pip install -r requirements.txt
   ```

3. Botを起動：

   ```bash
   python bot.py
   ```

## コマンド

### `/setup_button`

ロール付与ボタンを作成します（管理者のみ）。

| パラメータ | 必須 | 説明 |
|---|---|---|
| `role` | はい | 付与したいロール |
| `minutes` | いいえ | ロールを保持する時間（分）。デフォルト: 0 |
| `days` | いいえ | ロールを保持する時間（日）。デフォルト: 0 |
| `notify_channel` | いいえ | ロール剥奪時に通知するチャンネル |

`minutes` と `days` は併用可能です。どちらか1つ以上を指定してください。

**使用例：**

- `/setup_button @一時入室 minutes:10 #ロール通知` — 10分間付与、剥奪時に通知
- `/setup_button @見てる人 days:30` — 30日間付与、自動で剥奪
- `/setup_button @ゲスト days:1 minutes:30` — 1日30分間付与

### `/help`

Botの使い方を表示します。

## 注意事項

- Botのロールは、付与・剥奪対象のロールより上位に配置してください
- タイマー情報は `role_timers.json` に保存されます。Bot再起動後も剥奪処理は継続します
- 30秒ごとに期限切れチェックが行われるため、剥奪タイミングには最大30秒の誤差があります
